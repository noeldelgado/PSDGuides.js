psd-guides is a library to draw photoshop-like guides.

Download .zip file View on Github

Sample Usage

Include PSDGuides.js into your project.

– Custom guides example

new PSDGuides({
  canvasWidth : 1000,
  xGuides     : [20, "355 * 2", 250, 20],
  yGuides     : [50, "100 * 2", "250 * 2", "50 * 3"]
});

– Applying 960 grid system

12 column grid

new PSDGuides({
  canvasWidth : 960,
  xGuides     : [10, 60, 10]
});

16 column grid

new PSDGuides({
  canvasWidth : 960,
  xGuides     : [10, 40, 10]
});

24 column grid

new PSDGuides({
  canvasWidth : 960,
  xGuides     : [10, 20, 10]
});
